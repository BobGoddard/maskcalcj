To build and run...<BR>
1. CD into the src directory<BR>
2. javac -Xdiags:verbose maskcalc/*.java<BR>
3. jar -c -f maskcalc.jar -v -m MANIFEST.MF maskcalc/*.class maskcalc.png<BR>
4. java -jar maskcalc.jar
